VPN Manager

VPN Manager это openVPN клиент для телевизоров с ситемой WebOS

Требования: WebOS не ниже 4.0 +; наличие рут.
После установки приложения обязательно перезагрузите телевизор!
Внимание! В приложении используются бесплатные файлы конфигурации .ovpn (конфиг) найденные в интернете. Бесплатные конфиги найденные в интернете не всегда могут быть рабочими и скорость vpn соединения при работе с ними оставляет желать лучшего. Для качественного vpn соединения лучше использовать собственный файл конфигурации
